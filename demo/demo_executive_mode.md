## Sections To Highlight

- `What to Do Now`
- `What to Verify Next`
- `Key Uncertainties`
- `Suggested Next-Step Plan`

## Demo Angle

Use this mode when you want to show that the same underlying analysis can be compressed into a shorter memo. The key message is that this mode answers “what should we do next and why” without exposing every technical detail.
